# dk-app

## Introduction

Cette application a été créer dans le but de pouvoir gérer des cours de coréen en groupe.

## Structure

Dans ce projet est découpé en 2 partie, la partie front qui est actuellement disponible [ici](https://danbee-korean.com/), ainsi que la partie back qui est en cours de développement.

### Front

Le front end de l'application a été développé en utilisant NextJS. J'utilise donc aussi TailwindCSS ainsi que l'UI shadcn pour les composants de l'application.
Je ne suis à la base pas un développeur web, cependant il reste assez simple d'apprendre les concepts lié au développement web, j'ai donc à travers de ce projet, ainsi que de certaines expériences professionnels, pu améliorer mes compétences dans ce domaine.


### Backend

Le backend est une partie que je maitrise le mieux, et qui me fascine vraiment beaucoup de par sa faculté à construire une architecture de projet. J'ai déjà pu participé et concevoir par le passé plusieurs backend d'application que se soit pour des projets universitaires ou entreprises.
Nous utilisons ici:

- Environnement: NodeJS
- Langage: TypeScript
- API: Express
- Base de données: MySQL
- Session: express-session
- Cache: Redis
